Chapter 2. Moving the Player

2.1. Before You Start

The Rigidbody2D Class

This class is necessary to take advantage of 2D physics. You can control the object’s movement and the influence of gravity through this class.

For details, check out Rigidbody2D in the documentation.

The Input Class

This class processes controller input from keyboards, joysticks, etc. It can also access gyros and accelerometers in mobile devices.

For details, check out Input in the documentation.

“Attaching”

In this tutorial, you’ll be asked to “attach” things. This is the process of adding a component to a game object, which you’ll do often.

Writing The Script

Let's quickly add some code. Double-click the Player.cs file, and MonoDevelop will start up.

We will use Input.GetAxisRaw(string axisName) to link the arrow keys to the player movement.

Input.GetAxisRaw

When the left arrow is pressed, it will return -1, the right arrow will return 1, and when nothing is pressed, the result is 0. Use this only when you want to see if a keyboard key is pressed or not.

You can use Input.GetAxis(string axisName) when you want to get continuous values between zero and one, like on an analog stick.

When the arrow keys are pressed, we set the player’s velocity vector by assigning to rigidbody2D.velocity.

Velocity Vector??

This represents movement speed and direction in a single Vector.

This document provides more information on using vectors in Unity: Vector Cookbook

Below is the code for Player.cs.

Player.cs

using UnityEngine;
using System.Collections;

public class Player : MonoBehaviour
{
        // Movement Speed
        public float speed = 5;

        void Update ()
        {
                // Right, Left
                float x = Input.GetAxisRaw ("Horizontal");

                // Up, Down
                float y = Input.GetAxisRaw ("Vertical");

                // Get the direction of movement
                Vector2 direction = new Vector2 (x, y).normalized;

                // Assign the movement speed and direction
                rigidbody2D.velocity = direction * speed;
        }
}
