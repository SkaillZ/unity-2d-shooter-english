Chapter 12. Make Five Waves, and Add Scoring

Right now, there’s only a single Wave. Let’s raise that to 5, and also implement scoring.

12.1. Make a Wave

We’ll add four extra Waves. The easiest way to do this is to just duplicate the Wave prefab made in Chapter 7. Go ahead and place the enemies wherever you want. If you prefer, you can download these four example Waves, described below.

Download Wave.unitypackage

Wave 2

A large number of 4-HP enemies, that do not fire.

Wave 3

Enemies with 10 HP that fire their weapons.

Wave 4

Enemies that shoot, and appear from the left/right, with 10 HP.

Wave 5

These enemies are triple the size, with 100 HP, and fire a barrage of bullets just once.

Once the Waves are made, drag and drop them into the Waves property of the Emitter game object.

Play the game, and see that the waves cycle from 1 to 5, and back to 1 again.

If the game is too hard, feel free to adjust the difficulty. You can alter the enemies’ HP, heading, speed or even the number of ShotPosition game objects.

12.2. Implementing Scoring

Now let’s implement a “score,” the final element of our game.

We'll add both a current score indicator, as well as a high score.

Score Indicator

Set up a GUIText to show the score onscreen.

Create an empty game object, and name it Score GUI. Set the position of the transform to X 0 Y 0 Z 0.

In the Hierarchy window click the Create button and select GUI Text. This will create a game object with a GUI Text component included.

Make another one of those objects, and make both of them children of Score GUI. Name them Score and HighScore.

Now we can set up the Score and HighScore objects.

Set up the Transform’s Position, and the Text, Anchor, Font and Font Size of the GUIText to match that in Figure 12.1 and Figure 12.2.

Make a Script

Create Score.cs in the Scripts folder.

We can save the score using the "PlayerPrefs" class.

PlayerPrefs

This class is used for saving any game data you want.

For example, game progress, sound volume, high scores, rankings, and so on are all things you might save with PlayerPrefs.There are special restrictions on its use, so check PlayerPrefs in the documentation for details.

Score.cs

using UnityEngine;

public class Score : MonoBehaviour
{
        // The GUIText that shows the Score
        public GUIText scoreGUIText;

        // The GUIText that shows the High Score
        public GUIText highScoreGUIText;

        // Score
        private int score;

        // High Score
        private int highScore;

        // The key for saving with PlayerPrefs
        private string highScoreKey = "highScore";

        void Start ()
        {
                Initialize ();
        }

        void Update ()
        {
                // If the Score is higher than the High Score…
                if (highScore < score) {
                        highScore = score;
                }

                // Display both the Score and High Score
                scoreGUIText.text = score.ToString ();
                highScoreGUIText.text = "HighScore : " + highScore.ToString ();
        }

        // Return to the original game state
        private void Initialize ()
        {
                // Set the score to 0
                score = 0;

                // Retrieve the high score. If it can’t be received, use zero.
                highScore = PlayerPrefs.GetInt (highScoreKey, 0);
        }

        // Adding points
        public void AddPoint (int point)
        {
                score = score + point;
        }

        // Saving the High Score
        public void Save ()
        {
                // Save the high score
                PlayerPrefs.SetInt (highScoreKey, highScore);
                PlayerPrefs.Save ();

                // Return to the original game state
                Initialize ();
        }
}

Add the Score.cs you created to the Score GUI game object by dragging and dropping.

Now drag and drop the Score object to the Score GUIText, and the HighScore to the High Score GUIText.

Now the scores will be shown. Next we’ll add point values to the player’s score.

Give Point Values to the Enemies

We want to add points whenever an enemy is defeated. To do this we’ll add a point value to Enemy.cs.

We'll also make it so that points are added when the enemy explodes in OnTriggerEnter2D.

Enemy.cs

using UnityEngine;
using System.Collections;

public class Enemy : MonoBehaviour
{
        // Hit Points
        public int hp = 1;

        // Point Value
        public int point = 100;

        // Spaceship Component
        Spaceship spaceship;

        IEnumerator Start ()
        {

                // Get the Spaceship component
                spaceship = GetComponent<Spaceship> ();

                // Move in the negative direction of the Y-Axis
                Move (transform.up * -1);

                // If canShot is false, end the coroutine here.
                if (spaceship.canShot == false) {
                        yield break;
                }

                while (true) {

                        // Get the child objects
                        for (int i = 0; i < transform.childCount; i++) {

                                Transform shotPosition = transform.GetChild (i);

                                // Fire a bullet at shotPosition’s position/rotation.
                                spaceship.Shot (shotPosition);
                        }

                        // Wait for shotDelay seconds.
                        yield return new WaitForSeconds (spaceship.shotDelay);
                }
        }

        // Move the ship
        public void Move (Vector2 direction)
        {
                rigidbody2D.velocity = direction * spaceship.speed;
        }

        void OnTriggerEnter2D (Collider2D c)
        {
                // Get the Layer name
                string layerName = LayerMask.LayerToName (c.gameObject.layer);

                // Return immediately if the layer is not “Bullet (Player)”
                if (layerName != "Bullet (Player)") return;

                // Get the PlayerBullet’s Transform
                Transform playerBulletTransform = c.transform.parent;

                // Get the Bullet component
                Bullet bullet =  playerBulletTransform.GetComponent<Bullet>();

                // Deduct the hit points
                hp = hp - bullet.power;

                // Delete the bullet
                Destroy(c.gameObject);

                // if the hit points fall to 0 or less…
                if(hp <= 0 )
                {
                        // Get the Score component and add the points.
                        FindObjectOfType<Score>().AddPoint(point);

                        // Explode
                        spaceship.Explosion ();

                        // Delete the enemy
                        Destroy (gameObject);

                }else{

                        spaceship.GetAnimator().SetTrigger("Damage");

                }
        }
}

Now adjust the points for the enemies in each Wave prefab. Make the score dependent on the difficulty, so that enemies with more hit points are worth more.

Now play the game. When you defeat enemies, points are added, and the high score is updated in real time.
