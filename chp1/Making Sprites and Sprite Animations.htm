Chapter 1. Making Sprites and Sprite Animations

Sprite

Sprites provide the graphics for items and characters that appear in 2D games.

For detailssou, see the Sprite and SpriteRenderer documentation pages.

1.1. Use the Sprite Editor to Make Multiple Sprites From One Texture

In the Project View, open the Sprites/Spaceship folder and select Spaceship.

Figure 1.1: Using the mouse to select the Spaceship.

Review the Texture Import Settings

In 2D mode, textures will be imported as Sprites.

If Texture Type is set to Texture, change it to Sprite and click Apply as shown in Figure 1.2.

Figure 1.2: Setting the Texture Type to Sprite

To turn this one Texture into multiple Sprites, go to its Texture Import Settings, set Sprite Mode to Multiple, and Pixels To Units to 50.

Figure 1.3: Texture Import Settings for the Spaceship texture.

Pixels To Units

Unity measures length in Units. You can think of it as 1 unit to 1 meter.

“Pixels to Units” sets the ratio between screen pixels and game units.

When Pixels To Units Is 1

One game unit becomes the same size as 1 pixel.

When Pixels To Units Is 100

One game unit becomes the same size as 100 pixels.

Click the SpriteEditor button in Texture Import Settings to open up the SpriteEditor.

Figure 1.4: The Sprite Editor

SpriteEditor

Sprites can contain a single graphic, but it’s more efficient to pack many graphics into one. You can do this with the Sprite Editor.

For details, see the SpriteEditor documentation page.

Dividing Along a Grid

Let’s make multiple Sprites from a single Texture. In the upper right, click Slice.

Change the Type from Automatic to Grid.

Figure 1.5: Changing the Type in the Sprite Editor

Change the Pixel Size to X 48 Y 48.

Figure 1.6: Setting the Pixel Size

Click Slice, confirm that the lines properly frame the textures, and press the Apply button.

Figure 1.7: A grid is superimposed over the texture.

If your Project View looks like Figure 1.8, everything is OK.

Figure 1.8: The Spaceship resource in the Project View.

1.2. Animating the Player’s Ship

Let's use Unity to make the animation shown in Figure 1.9.

Figure 1.9: A GIF of the desired animation.

Making Animation Controller and Animation Clip Files

AnimatorController and AnimationClip files are created when dragging and dropping multiple Sprites onto the Scene View.Drag the set of items from Spaceship_0 to Spaceship_3 and drop them into the Scene View.

Figure 1.10: Dragging and dropping the first Spaceship into the scene.

Now, a dialog box will appear where you can name and save the animation file.

Save it as Normal in the Animations/Player folder.

Figure 1.11: Saving to the Animations/Player folder.

Find the game object named Spaceship_0, and change its name to Player.

Figure 1.12: The new Player game object.

Figure 1.13: The new Player resource.

Changing the Name of a Game Object

There are two ways to change the name of a game object.

1. Changing the name in the top of the Inspector

When selecting a game object, the Inspector displays various information about it. At the top, its name is an editable text field. Change the name, and press Enter.

Figure 1.14: Changing a game object’s name in the Inspector

2. Changing the name in the Hierarchy

Select a game object in the Hierarchy, press Enter, and the name will become editable. Change the name, and press Enter.

Figure 1.15: Changing a game object’s name in the Hierarchy

Okay, our sprite animation is already done.

Press the Play button, and see what happens!

Figure 1.17: Creating the second Spaceship animation.

Set the name of this one to Enemy.

Figure 1.18: Naming the enemy game object.

Figure 1.19: Naming the enemy resource

1.4. Making the Bullets

Open the Bullet in the Sprite Editor, and set its Sprite Mode to Multiple.のSprite ModeをMultipleにしてSprite Editorを開きましょう。

Set the Type to Grid, and the Pixel Size to X 16 Y 16.

Figure 1.20: Configuring the Bullet sprite.

Slice the sprite and press Apply.

1.5. The Explosion Animation

Let's make an animation that looks like this:

Figure 1.21: A GIF of the desired animation.

Open the Explosion in the Sprite Editor, and set its Sprite Mode to Multiple.

Set the Type to Grid, and the Pixel Size to X 80 Y 80.

Slice the sprite and press Apply.

Figure 1.22: Slicing the Explosion sprite.

In the same way as the two ships, select Explosion_0 through Explosion_10, and drop them into the scene, creating an AnimationClip and AnimatorController. Name the Animation Explode.

Figure 1.23: Creating the Explosion game object.

Figure 1.24: Creating the Explosion resource.

Creating a Prefab

Let's take the Player, Enemy, and Explosion that we made, and create Prefabs out of them.

Make a folder named Prefabs, and drag each game object, one by one, into the folder.

Figure 1.25: Drag each of these game objects, one by one, into the Prefabs folder.

Once you’ve made those three prefabs, you can delete the original game objects out of the scene.

Right-click the game objects in the Hierarchy, and select Delete from the menu that appears.

Prefab

A game object that’s been saved as an asset is called a Prefab.

Game Object -> Prefab

Easily make game objects with the same contents.

Figure 1.26:

For details, see the Prefabs section of the documentation.
