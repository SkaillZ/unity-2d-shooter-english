Before You Start

1.1 Requirements

Unity

Download Unity

Tutorial Project

Download the Project File for this Tutorial

1.2 Get Acquainted With Unity

Have you gone through the basic Unity tutorials?

Before you take on this tutorial, first you’ll want to learn the basics about Unity.

1.3 Starting Up Unity

Unarchive the previously-downloaded ShootingGame.zip and open the included Unity project.

Figure 1.1: Unity, after starting up.

Launching Unity

On the Mac, Unity.app is installed to your Applications folder. On Windows, double-click the shortcut to Unity.exe that was placed on your Desktop.

Figure 1.2: The Unity application, as it appears in Finder on the Mac.

Selecting a Project to Open

If the Project Wizard appears, click the “Open Other…” button.

If the Project Wizard does not appear when starting Unity…

Click “File -> Open Project…” and the Project Wizard should open.

1.4 Set Up Your Tutorial Environment

To make the tutorial go smoothly, make your Unity configuration match the image in Figure 1.3.

Figure 1.3: Unity environment for this tutorial.

Changing the Layout

Changing Unity’s Layout

Let’s change the layout, to get the right “look” in Unity. Click the Layout button in the upper right and select 2 by 3.

Figure 1.4: Be sure to set the layout to 2 by 3.

Changing the Project View Layout

Click the " ≡ " glyph in the upper right of the Project View, and select One Column Layout.

Figure 1.5: Selecting the “One Column Layout” option in the Project View

Changing the Build Target

This time, we’re going to make a WebPlayer game.

Select “File -> Build Settings…” and the Build Settings window will open.

By default, PC, Mac & Linux Standalone should have a Unity logo on its side. You can change the platform by following the process in Figure 1.6.

図1.6: ① Select this ② Switch the selected platform

Changing the Game Screen Size

Change the game screen size to 600 x 450. Open “Edit -> Project Settings -> Player” and the Player Settings will open.

Figure 1.7: Accessing the Player Settings.

Set the Default Screen Width to 600, and the Default Screen Height to 450.

Figure 1.8: Player Settings, as it appears when selecting the Web Player.

Next, set the Game View to match the screen size. Click the button on the upper-left of the game view, and select Web (600x450).

Figure 1.9: Setting the Game View size.

Check for 2D Mode

Check to make sure that the 2D button at the top of the Scene View is pressed.

1.5 Environment Setup Complete

You’ve completed the first steps in setting up Unity. This should make it easy for first-timers to adjust to the Unity environment.

This setup is designed to fit this tutorial, you don’t necessarily have to set it up this way.

After completing the tutorial, feel free to customize Unity to fit your working style.
